# retrographics
This is an example of how i made PS1 styled graphics in raylib-cs

## How to compile:

First you need to install the .NET 6 SDK

To get the dependencies use:
``
dotnet restore
``

To build the executable use:
``
dotnet build
``

To run the code use:
``
dotnet run
``
